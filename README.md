# Cursor Test Components

A React component library built from Figma designs with Code Connect integration.

## Components

This library includes the following components:

- **Button** - Primary, Secondary, and Hyperlink button variants with optional icons
- **Product Brand Rating** - Product branding with star ratings and wishlist functionality
- **Icon with Text** - Icon and text combinations
- **Card Selector** - Selectable card components with icons
- **Size & Color Selector** - Product size and color selection dropdowns
- **Product Price** - Price display with strikeout and discount options
- **Star Rating** - Star rating component with rating counts
- **Thumbnail Row** - Row of product thumbnails (3 or 4 items)
- **Angle Right** - Right navigation arrow in multiple sizes
- **Angle Left** - Left navigation arrow in multiple sizes
- **Product Media Section** - Product image gallery with thumbnails and navigation
- **Thumbnail** - Individual thumbnail component
- **Utility Links** - Utility link icons (1-6 items)
- **Links** - Link components with icons and text
- **Nav Menu** - Navigation menu with multiple items (1-8 items)
- **Menu** - Individual menu item component
- **Breadcrumbs** - Breadcrumb navigation component
- **Header Base** - Header component with logo alignment options
- **Dropdown** - Dropdown select component

## Component Showcase

View all components in the interactive HTML showcase page:
- Open `components.html` in your browser to see all 20 components with their props, descriptions, and file paths
- Features search functionality to quickly find components
- Shows Code Connect mapping status for each component

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Technology Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite

## Code Connect

All components are mapped to Figma designs using Code Connect for seamless design-to-code synchronization.
